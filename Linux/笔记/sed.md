# 参数 #
-f 指定脚本文件

# 模式 #
匹配行/动作[/pattern1/replacement/flags]

# 匹配行 #
1. 1 第一行
2. 1,10 第1到第10行
3. 1,$ 第1行到最后1行
4. /regexp/ 该行要满足正则表达式
5. P1,P2 选中 P1~P2 的这些行, P1和P2可以是数字, 也可以是正则表达式

# 分组命令 #
删除1~10行中的空行
并且将 priec 替换成 price
```
1,10{
	/^$/d
	s/priec/price/g
}
```

# 动作 #
d 删除
s 替换
p 打印
