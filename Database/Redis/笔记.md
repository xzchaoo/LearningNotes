1. 在a处执行rewriteaof, 使得aof文件变小
2. 关闭b
3. 将a的aof文件复制到b处, 替换b的aof文件
4. 
4. 启动b, 此时b应该会使用新的aof进行数据恢复
## 利用RDB ##
b都应该使用RDB, 而没有AOF
1. 在a出执行save, 导出最新的RDB
2. 关闭b
3. 将a的RDB文件复制到b出, 替换b的rdb文件
4. 启动b, ...
