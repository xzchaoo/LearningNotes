## 从文件加载数据到表 ##

文件内容这样:

2	xzc2	22

注意字段之间用\t隔开 而不是空格!

## NULL值的处理 ##
select 1 = null, 1!=null ,1 is null, 1 is not null;
结果是 null null 0 1
将1换成0 结果页是一样
### 几个结论 ###
1. 有null参与的数学运算(+-*/%)和比较全部是null
2. group by xxx, xxx is null的记录会被分为一组
3. null 不会参与聚合运算
