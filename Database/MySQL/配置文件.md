sql-mode="某个模式" 如果是空字符串那就表示没有
也可以在mysql里执行
SET GLOBAL sql_mode = 'modes';
SET SESSION sql_mode = 'modes';
SELECT @@GLOBAL.sql_mode;
SELECT @@SESSION.sql_mode;
http://dev.mysql.com/doc/refman/5.7/en/sql-mode.html
