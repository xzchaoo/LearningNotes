# task 系列 #
## 创建taks ##

```
//立即结果
var task = Task.FromResult("a");
task.ContinueWith(task0 =>
{
    Console.Out.WriteLine(task0.Result);
});
```


```
PrintTid();
//会放到线程池里去执行
var task = Task.Run(() =>
{
    PrintTid();
    return "asdf";
});
task.ContinueWith(task0 =>
{
    Console.Out.WriteLine(task0.Result);
});
Thread.Sleep(1000);
```

```
var task = new Task(工作函数);
task.Start();
task.wait();
```
