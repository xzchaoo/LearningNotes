热字符串
可以起到"展开"的作用, 比如输入 btw 然后按下任意的结束字符(比如回车 空格 tab键等)
就会展开成 By the way
格式是:
::热字符串::

热字符串也可以用于触发一个动作:
::btw::
MsgBox "By the way"
Return


变量赋值方式
a=b 等价于 a:="b"
a=%b% 等价于 a:=b

a:=b+4/2+sqrt(9)
a=%b% some text %c%.
a=b " some text " c

当使用 command 的时候通常会和%配合使用
输入变量名和输出变量名除外

什么时候不使用%
1. 变量名和输出变量名时
2. 赋值操作符的左边 a = %b%
3. If Var1 < %Var2%
